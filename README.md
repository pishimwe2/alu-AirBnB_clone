\# AirBnB Clone Project



\## Description of the project



The AirBnB Clone project is a command-line application that recreates the core functionality of the AirBnB platform. It allows users to create, view, update, and delete objects, while storing data persistently in JSON format.



This project helps practice:

\- Python programming

\- Object-Oriented Programming (OOP)

\- Command interpreter development

\- File storage and serialization

\- Team collaboration using Git and GitHub



\## Description of the command interpreter



The command interpreter is an interactive shell that allows users to manage objects through commands.



\### How to start it



Run the following command:



```bash

./console.py

```



Or:



```bash

python3 console.py

```



\### How to use it



After starting, you will see the prompt:



```bash

(hbnb)

```



Type commands in this format:



```bash

(command) \[arguments]

```



\### Examples



Create a new object:



```bash

(hbnb) create BaseModel

```



Show an object:



```bash

(hbnb) show BaseModel <id>

```



Update an object:



```bash

(hbnb) update BaseModel <id> name "My First Model"

```



Delete an object:



```bash

(hbnb) destroy BaseModel <id>

```



Quit the interpreter:



```bash

(hbnb) quit

```



Or:



```bash

(hbnb) EOF

```
